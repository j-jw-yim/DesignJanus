# DesignJanus: Project Brief

## Project Name

**DesignJanus**

---

## One-Sentence Thesis

Build a human-centered benchmark for perceptual multi-view consistency in 2D→3D outputs.

---

## Research Questions

1. **Which inconsistency types matter most to people?**
2. **Which ones reduce design usefulness most?**
3. **Which simple automatic metrics best track human judgment?**

---

## Target Final Deliverables

| Deliverable | Description |
|-------------|-------------|
| **Benchmark dataset** | 20 prompts × 3 methods × 6 views; standardized format |
| **Annotation interface** | Streamlit app for human ratings; CSV export |
| **Analysis notebook** | Jupyter notebook with plots, correlations, ranked importance |
| **Short report / portfolio page** | Problem, method, findings, key insight |

---

## Scope (Locked)

- 20 prompts across 4 object groups
- 3 methods
- 6 views per sample
- 5 inconsistency categories
- Pilot with 5–8 people before full study
